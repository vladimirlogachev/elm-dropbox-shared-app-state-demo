# User Manual

## What is this?

A demo application showing shared app state via Dropbox. Multiple clients can send chat messages that are synchronized through a shared JSON file in Dropbox.

## Getting started

1. Click "Open app" on the landing page.
2. Authorize with your Dropbox account.
3. Send messages in the Chat page — they sync across all clients connected to the same Dropbox app folder.

## Pages

- **Chat** — Send and receive messages. Messages are synced via Dropbox.
- **Help** — This manual.
- **Settings** — Sign out or reinitialize app state.
